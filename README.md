# The morp profile

Welcome to my morp profile dev stuff, here you can find the source code for all the code, html and stuff (no server because private)

## Core features

- Cool little minigame to start off in the main page
- Interactland, a place where you can upload any image to show off to others, leave your mark!
- Interesting sound effects yknow, ding and buzzzzzzzz
- Written in typescript + tailwind + vite
- custom build script that automatically updates the morp profile page live!

This is just a profile page so there isn't much i wanted to put on it but interact land is pretty cool imo, check out the source code for it!

## Working with morp

As the morp profile page is just the index page with inline styling and javascript, I had to be very finiky with it, creating a script that would inline the complied typescript and tailwind into the html which wasn't really that hard but then having an entirely different game with its own logic was tuff as morp kept rejecting what I was making but in the end it all works!
